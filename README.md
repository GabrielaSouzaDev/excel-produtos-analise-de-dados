# 📊 Excel Produtos - Análise de Dados
Projeto desenvolvido para análise e manipulação de dados utilizando Excel, com foco em organização, interpretação e visualização de informações de forma prática e eficiente.

## 🚀 Funcionalidades
Importação e organização de dados Tratamento de informações Análise exploratória de dados Automatização de tarefas no Excel

## 🛠️ Tecnologias Utilizadas
Microsoft Excel Python Pandas Openpyxl

## 📂 Estrutura do Projeto
excel-analise-de-dados/
│

├── excel.py 

├── README.md 

└── ocupacao.xlsx

## ▶️ Como utilizar
Clone o repositório: git clone https://github.com/GabrielaSouzaDev/excel-analise-de-dados.git Abra o arquivo .xlsx no Microsoft Excel. Instale a biblioteca pandas e openpyxl

## 📈 Objetivo do Projeto
O objetivo deste projeto é demonstrar técnicas de análise de dados utilizando Excel, aplicando recursos de organização, visualização e interpretação de dados para apoiar tomadas de decisão.

## 📚 Aprendizados
Durante o desenvolvimento deste projeto foram praticados conceitos como:

Limpeza e tratamento de dados Visualização de dados Análise exploratória Automação de processos no Excel
